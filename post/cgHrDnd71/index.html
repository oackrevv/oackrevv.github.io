DDOS攻击

2022-10-28

如何确认是否受到DDOS攻击？
netstat -ntu | awk '{print $5}' | cut -d: -f1 | sort | uniq -c | sort -n
执行后，将会显示服务器上所有的每个IP多少个连接数。如图：

很明显，前面的数字就是连接数！连接数过大就说明不正常了。

1.服务器内部安装 iftop 流量监控工具

yum install iftop -y

 2.运行下面命令查看流量占用情况

iftop -i eth1 -P
