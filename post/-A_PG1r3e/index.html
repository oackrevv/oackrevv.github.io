thinkphp5 提示错误 A non well formed numeric value encountered

2022-10-28

方法：
第一步:将字段类型设置成datetime

第二步：在数据库配置文件：database.php里面开启自动写入时间戳字段
将
// 自动写入时间戳字段
'auto_timestamp' => false,
改成
'auto_timestamp' => 'datetime',
或
// 时间字段取出后的默认时间格式
'datetime_format' => "Y-m-d H:i:s",
改成
'datetime_format' => false,
